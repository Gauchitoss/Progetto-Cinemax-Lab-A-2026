Per Installare:
Aprire il terminale del computer e eseguire: https://github.com/Gauchitoss/Progetto-Cinemax-Lab-A-2026.git
Così facendo il terminale scaricherà l'intero progetto creando una nuova cartella (che si chiamerà come il tuo repository). Questa nuova cartella verrà creata all'interno della cartella in cui si trova il tuo terminale in quel momento.  
In alternativa, se non hai Git installato, si può cliccare sul pulsante verde code in alto a destra sulla schermata della repository su GitHub, selezionare Download ZIP, una volta scaricato il file aprire esplora file e premere con il testo destro del mouse sul file .zip scaricato e premere su estrai tutto. 

Per Eseguire:
Per avviare CineMax, aprire un terminale o prompt dei comandi, spostarsi nella cartella principale del progetto, se ad esempio la vostra cartella si chiama LaboratorioA, il comando sarà: cd C:\percorso_file\LaboratorioA
Per eseguire il programma digitare a terminale: java -jar bin/Laboratorio_A.jar
